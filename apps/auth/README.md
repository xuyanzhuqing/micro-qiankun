#### 启动工程

```bash
pnpm start
```

#### 打包工程
```bash
pnpm build
```

本工程采用 [create-react-app](https://create-react-app.dev/docs/getting-started) 构建, 使用 [craco](https://craco.js.org/docs/getting-started/) 自定义配置 [webpack v5](https://www.webpackjs.com/configuration/)
